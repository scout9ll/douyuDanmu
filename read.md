# django-react 搭建流程

## 创建 Django 生成 api 应用

## 在 django 项目中搭建 react 前端

### 创建前端目录

- 组件资源 原始 react
- 静态 打包目标 js
- 模板 目标 Html

### 添加所需工具

- `npm init -y` 模块依赖文档
- `npm i -D webpack webpack-cli`打包工具
- `npm i -D @babel/core babel-loader @babel/preset-env @babel/preset-react babel-plugin-transform-class-properties`babel 编译器
- `npm i react react-dom prop-types` react 模块
